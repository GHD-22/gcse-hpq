# GCSE HPQ: UK Climate Policy Analysis

**Analysing the Zero Emission Vehicle (ZEV) Mandate and Contracts for Difference (CFD) Scheme**

---

## Project Overview

This repository contains my GCSE Higher Project Qualification (HPQ) research project, 
analysing two UK Government climate policies:

1. **The ZEV Mandate** — a regulatory obligation requiring manufacturers to sell a rising 
   percentage of zero emission vehicles each year
2. **The Contracts for Difference (CFD) Scheme** — a market-based financial mechanism 
   supporting investment in renewable electricity generation
